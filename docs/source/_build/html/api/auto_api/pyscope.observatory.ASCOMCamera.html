SetImageDataType()[source]#

Determine the data type of the image array based on the MaxADU property.

This method is called automatically when the ImageArray property is called if it has not already been set (initializes to None). It will choose from the following data types based on the MaxADU property:

  • numpy.uint8 : (if MaxADU <= 255)

  • numpy.uint16 : (default if MaxADU is not defined, or if MaxADU <= 65535)

  • numpy.uint32 : (if MaxADU > 65535)
