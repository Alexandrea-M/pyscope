These programs are from the PuTTY SSH client package, downloaded from:

http://www.chiark.greenend.org.uk/~sgtatham/putty/download.html

They are called in order to transfer images and telrun files between 
the telescope computer and the Iowa server.

plink.exe: an SSH client that can be called from the command line
           (contrast with "putty.exe", which runs as a GUI and cannot
            be controlled from the command line)
pscp.exe: a secure copy client; similar to the "scp" command in Linux
psftp.exe: a secure ftp client; similar to the "sftp" command in Linux
puttygen.exe: a tool for generating public/private key pairs to enable
              secure automated logins without passwords
